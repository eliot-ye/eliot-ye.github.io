# 格式化数字

## 从右到左，每三位数加上 `","`

```js
{
  get() {
    return this.value.replace(/(\d)(?=(?:\d{3})+$)/g, "$1,");
  },
  set(data) {
    this.value = data.replace(/,/g, "");
  }
},
```

## 从左到右，每四位数加上 `" - "`

```js
{
  get() {
    return this.value.replace(/(\w{4})(?=\w)/g, "$1 - ");
  },
  set(data) {
    this.value = data.replace(" - ", "");
  }
},
```
