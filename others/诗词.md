# 诗词

## 丑奴儿 · 书博山道中壁

（南宋）辛弃疾

```md
少年不识愁滋味，爱上层楼。爱上层楼，为赋新词强说愁。

而今识尽愁滋味，欲说还休。欲说还休，却道天凉好个秋！
```

## 侠客行

（唐） 李白

```md
赵客缦胡缨，吴钩霜雪明。
银鞍照白马，飒沓如流星。
十步杀一人，千里不留行。
事了拂衣去，深藏身与名。
闲过信陵饮，脱剑膝前横。
将炙啖朱亥，持觞劝侯嬴。
三杯吐然诺，五岳倒为轻。
眼花耳热后，意气素霓生。
救赵挥金锤，邯郸先震惊。
千秋二壮士，烜赫大梁城。
纵死侠骨香，不惭世上英。
谁能书阁下，白首太玄经。
```
