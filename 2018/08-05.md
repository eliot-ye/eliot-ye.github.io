# 关于checkbox的展示应用

## 样式，替代法

```html
<label class="checkbox">
  <input type="checkbox" id="">
  <span></span>
</label>
```

```scss
.checkbox{
  vertical-align: middle;
  input[type=checkbox]{display: none}
  span{
    display: inline-block;
    width: 16px;
    height: 16px;
    border: 1px solid #A38601;
  }
  input[type=checkbox]:checked+span{
    background: url(../img/checked.png) no-repeat center/12px auto;
  }
}
```

## 开关switch

```html
<label class="switch">
  <input type="checkbox" id="is-discount">
  <span class="switch-bg">
    <span class="switch-rang"></span>
  </span>
</label>
```

```scss
.switch{
  display: inline-block;
  width: 1rem;
  height: 0.6rem;
  border-radius: 0.4rem;
  overflow: hidden;
  position: relative;
  *{transition: all 0.2s}
  .switch-bg{
    display: block;
    height: 0.6rem;
    background: #E6E6E6;
    padding: 0.05rem;
    .switch-rang{
      display: inline-block;
      width: 0.5rem;
      height: 0.5rem;
      border-radius: 0.3rem;
      background: #fff;
      position: absolute;
      left: 0.05rem;
    }
  }
  input[type=checkbox]{display: none}
  input[type=checkbox]:checked+.switch-bg{
    background: rgba(32,193,132,1) !important;
    .switch-rang{
      left:#{1rem - 0.55rem} !important;
    }
  }
}
```
