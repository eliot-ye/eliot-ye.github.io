## CSS

### stylelint

 CSS/SCSS/Less 的代码规范性检查

### Live Sass Compiler

实时编译 sass ,需要配置（基于 Live Server）

```js
"liveSassCompile.settings.formats":[
  // You can add more
  {
    "format": "compressed",// 压缩格式
    "extensionName": ".min.css",// 编译后缀名
    "savePath": "./css"// 编译保存的路径
  }
],
"liveSassCompile.settings.autoprefix": [
  // 自动加前缀的目标浏览器
  "> 1%",
  "last 2 versions",
  "not ie <= 8"
],
```
